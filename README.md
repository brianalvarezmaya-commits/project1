## Descripción
Este proyecto consiste en modificar un sitio web a gusto o nesesidad y hacer sus commits.
Fue desarrollado como parte del curso Desarrollo Web Fullstack.

El sitio permite:
- Navegar entre secciones
- Visualizar contenido informativo
todo referente a un emprendimiento


## Tecnologías Usadas
- HTML5
- CSS
- Git & GitHub
